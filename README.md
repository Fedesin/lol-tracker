# lol-tracker

Proyecto Laravel vacío con Filament y PostgreSQL para práctica de desarrollo.

## Requisitos

- PHP >= 8.2
- Composer
- PostgreSQL instalado y corriendo
- Node.js y npm (para assets y Filament)
- Git
